# Проєкт: Аналітична система для аналізу авіарейсів у США за допомогою PySpark

## Етап Видобування (ETL)

| Учасник | Відповідальність |
|----------|------------------|
| Мельниченко Марія | ETL для вимірів marketing_airline, operating_airline, airport |
| Владика Рената | ETL для вимірів date, time |
| Брездень Софія | ETL для факт таблиці FactFlight |

---

## Етап Трансформації (Data Transformation)

### Мельниченко Марія  
Категоріальна статистика і темпоральний аналіз, 6 бізнес-питань:

#### 1. FILTERS
- Які рейси були скасовані (Cancelled = True) у 2020 році під час пандемії COVID-19?

#### 2. JOIN
- Які маркетингові авіакомпанії використовували найбільше різних операційних авіакомпаній для виконання своїх рейсів?

#### 3. GROUP BY
- Яка загальна дистанція польотів по кожній операційній авіакомпанії за рік?  
- Яка середня дистанція польотів по днях тижня для кожного року?

#### 4. WINDOW FUNCTIONS
- Який відсоток затримок кожної авіакомпанії від загальної кількості затримок по місяцях?  
- Які топ-3 найбільш затримані рейси для кожної авіакомпанії в кожному кварталі?

---

### Владика Рената  
Загальна статистика датасету, 6 бізнес-питань:
#### 1. FILTERS
-  Скільки рейсів виконувала авіакомпанія Delta Airlines?
-  Які рейси з tail_number N789SK мали затримку у 2021 році?

#### 2. JOIN
-  Які аеропорти в штаті Каліфорнія мають найбільше затримок за назвами міст?
-  Які міста (city_name) мали найбільше direct connections з Atlanta у 2022 році?

#### 3. GROUP BY
- Яка середня затримка вильоту по кварталах для кожного року?

#### 4. WINDOW FUNCTIONS
-  Яка різниця затримки поточного рейсу порівняно з попереднім рейсом того ж tail_number?

---

### Брездень Софія  
Числова статистика — основні числові показники, кореляційний аналіз, аналіз розподілів, 6 бізнес-питань:

#### 1. FILTERS
- Скільки рейсів було скасовано в зимові місяці (грудень, січень, лютий) за всі роки?  
- Скільки нічних рейсів (після 22:00) прибули вчасно?

#### 2. JOIN
- Скільки рейсів виконувала кожна операційна авіакомпанія (Dim_Operating_Airline) у вихідні дні по місяцях?

#### 3. GROUP BY
- Яка середня тривалість польоту (air_time) по періодах дня (ранок/день/вечір/ніч)?  
- Який відсоток скасованих рейсів по штатах відправлення?

#### 4. WINDOW FUNCTIONS
- Яка середня затримка вильоту по кожному аеропорту з урахуванням останніх 7 днів?
